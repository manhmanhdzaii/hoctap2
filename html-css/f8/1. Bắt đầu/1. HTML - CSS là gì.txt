 - HTML là viết tắt của Hypertext MarkUp Language. Nó giúp người dùng tạo và cấu trúc các thành phần trong trang web hoặc ứng dụng, phân chia các đoạn văn, heading,link..

 - CSS là ngôn ngữ tạo phong cách cho trang web - Cascading Style Sheet language. Nó dùng để tạo phong cách và định kiểu cho những yếu tố được viết dưới dạng
ngôn ngữ đánh dấu, như là HTML.
